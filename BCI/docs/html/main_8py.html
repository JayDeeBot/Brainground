Detailed Description

Entry point for running the full BCI system.

This script launches the EEG GUI in a separate process, waits for user input, then coordinates EEG data acquisition, filtering, epoching, and asymmetry score processing using multiprocessing. It connects the GUI, DataAquisition, and ScanProcessing components into a functional pipeline.

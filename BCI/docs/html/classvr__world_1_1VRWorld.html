Detailed Description

Creates a 3D VR environment using Panda3D with a dynamic skybox and lighting system.

The VRWorld class sets up a Panda3D scene with a skybox model, ambient and directional lighting, and real-time lighting control based on an external score file (e.g., from EEG analysis). It continuously rotates the skybox for visual immersion and adjusts lighting dynamically.

Monitors an external file for lighting control values and updates the scene lighting.

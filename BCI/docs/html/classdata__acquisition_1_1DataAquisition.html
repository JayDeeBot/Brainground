Detailed Description

Handles real-time EEG data acquisition from an EDF file, visualization, and communication with processing module.

This class reads EEG data from an EDF file using the MNE library, allows channel selection, visualizes the data in real-time using Matplotlib, and sends processed EEG data chunks to the ScanProcessing class through a multiprocessing queue.
